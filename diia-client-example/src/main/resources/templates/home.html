This is a simple example of DIIA Client SDK usage

There are several scenarios what you can do with documents

  • Validating document by barcode
  • Requesting document by barcode
  • Requesting document by deeplink

Diia ID operations:

  • Sign document(s)
  • Authorization by Diia ID

Also you can manipulate branches

  • View all registered branches
  • Create new branch
  • Delete branch

And offers, created for branches

  • View all offers on the branch
  • Create new offer on the branch
  • Delete offer
